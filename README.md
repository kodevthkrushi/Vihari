# Vihari Project
“Every journey begins with Vihari.”

This is a travel listings web app built using **Node.js**, **Express**, **MongoDB**, and **EJS**.  
It allows users to create, view, and manage travel listings.

## Features
- Create new travel listings  
- View listing details  
- Edit or delete listings  
- Responsive layout with Bootstrap

## Tech Stack
- Node.js  
- Express  
- MongoDB (Mongoose)  
- EJS Templates  
- Bootstrap
